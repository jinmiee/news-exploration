# 📰 뉴스대탐험 (News Exploration)

실시간 뉴스·유튜브 이슈 분석 및 트렌드 시각화 플랫폼

뉴스와 유튜브 데이터를 기반으로 실시간 이슈를 수집·분석하고,
사용자가 현재 주목받는 이슈와 감정 흐름, 연관 키워드, 주간 트렌드를 한눈에 확인할 수 있도록 제작한 데이터 기반 뉴스 분석 플랫폼입니다.

조회수·언급량·중요도를 기반으로 뉴스 랭킹을 생성하고,
감정 분석 및 연관어 분석을 통해 단순 뉴스 나열이 아닌 “이슈 흐름 분석”을 목표로 한 서비스입니다.

# ⚙️ 데이터 파이프라인 상세

## 1. 데이터 수집 및 전처리

### YouTube API 기반 데이터 수집

* 6개 뉴스 채널 기반 기사 데이터 수집
* 기사 제목, URL, 조회수, 업로드 시간, 채널명, 썸네일, 댓글 데이터 확보
* Airflow 스케줄링 기반 오전 11시 / 오후 11시 자동 수집

### 텍스트 전처리

* clean_title() 기반 방송사명, 날짜, 특수문자 제거
* Okt 형태소 분석기를 활용한 명사/동사/형용사 추출
* 불용어 제거 및 핵심 키워드 기반 텍스트 정제

### 중복 기사 제거

* TF-IDF 및 KoBERT 기반 유사도 계산
* K-Means 클러스터링 기반 유사 기사 그룹화
* 중복 기사 제거 후 대표 기사 선정

---

## 2. 데이터 저장 및 관리

### MongoDB

* 뉴스 데이터
* 댓글 데이터
* 감정 분석 결과
* 연관어 분석 결과 저장

### SQLite

* 회원 정보
* 사용자 찜 데이터
* 서비스 관리 데이터 저장

### Airflow DAG 자동화

* 뉴스 수집 → 전처리 → 분석 → MongoDB 적재 자동화
* 스케줄링 기반 데이터 최신성 유지
* 차트 및 주간 이슈 자동 갱신

---

## 3. 분석 및 모델 적용

### 감정 분석

* HuggingFace 기반 BERT 감정 분석 모델 활용
* 긍정/부정/중립 감정 비율 시각화
* TF-IDF 기반 핵심 감정 키워드 추출
* 댓글 기반 사회적 반응 분석

### 연관어 분석

* KoSBERT 및 KoSimCSE 기반 유사도 계산
* MinHash 기반 유사 키워드 탐색
* 네트워크 그래프 기반 연관 키워드 시각화
* 정적/동적 오타 보정 기능 구현

### 뉴스 선정 알고리즘

* TF-IDF 기반 핵심 키워드 추출
* TF-IDF + KoBERT 유사도 4:6 비율 결합
* K-Means 클러스터링 기반 뉴스 그룹화
* 조회수 가중치를 반영한 대표 기사 선정
* 최종 Top 10 뉴스 자동 선정

### 중복 기사 처리

* 텍스트 유사도 기반 기사 그룹화
* 클러스터링 기반 대표 기사 선정
